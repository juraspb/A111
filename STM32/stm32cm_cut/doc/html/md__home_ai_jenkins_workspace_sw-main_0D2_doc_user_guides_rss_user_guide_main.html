User guides

User guides include a getting started guide for each of Acconeer's services and detectors. They describe how to get started with an easy example to retrieve data from the sensor which can be used in various applications. It is recommended to use the Exploration Tool for better understanding of the output data from the services.

Acconeer provides four different services in RSS

Envelope Service

IQ Service

Power Bins Service

Sparse Service

Acconeer also implements detectors which utilize the services to process the data for typical use cases

Distance Peak Detector

Obstacle Detector

Presence Detector
