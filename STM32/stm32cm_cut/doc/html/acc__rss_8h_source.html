22 /**
23  * @brief Activate the Acconeer Radar System Software, RSS
24  *
25  * A HAL struct containing integration functions (such as 'wait_for_interrupt', 'mem_alloc' and 'log')
26  * needed by RSS must first be populated and then sent in. See 'acc_definitions.h' for a full list
27  * of functions needed.
28  *
29  * This function must be called before any other RSS function. If it is not, or it failed,
30  * calling any other RSS function is undefined behaviour.
31  *
32  * @param[in] hal Reference to a HAL struct containing integration functions that is needed by RSS
33  * @return True if RSS activated successfully
34  */
