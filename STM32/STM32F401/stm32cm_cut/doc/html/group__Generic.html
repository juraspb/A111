◆ acc_service_profile_enum_t

enum acc_service_profile_enum_t

Service profile.

Each profile consists of a number of settings for the sensor that configures the RX and TX paths. Profile 1 maximizes on the depth resolution and profile 5 maximizes on radar loop gain with a sliding scale in between.
