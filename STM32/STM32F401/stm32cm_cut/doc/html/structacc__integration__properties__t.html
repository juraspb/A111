Detailed Description

This struct contains information about board properties that are needed by RSS.

sensor_count is the maximal sensor ID that the integration layer supports. This value must not exceed ACC_SENSOR_ID_MAX.

max_spi_transfer_size is zero or the maximal buffer size that is supported by the implementation of acc_hal_sensor_transfer_function_t. Use a zero value to indicate that there is no restriction. Otherwise, this value must not be smaller than ACC_SPI_TRANSFER_SIZE_REQUIRED.
